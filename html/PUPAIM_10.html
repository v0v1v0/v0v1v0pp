floryhuggins.LM R Documentation

Flory-Huggins Isotherm Linear Analysis

Description

Flory-Huggins isotherm model describes the degree of surface coverage characteristics of the adsorbate on the adsorbent. It describes the nature of the adsorption process regarding the feasibility and spontaneity of the process. The theory of the Flory-Huggins provides the mathematical model for the polymer blends' thermodynamics.

Usage

floryhuggins.LM(Ce, theta)

Arguments

Ce

the numerical value for the equilibrium capacity
theta

is theta fractional surface coverage

Value

the linear regression, parameters for Flory-Huggins isotherm, and model error analysis
