filenames_chelsa_trace21k R Documentation

Generate file names to download the chelsa trace 21k

Description

This function creates a vector of paths needed to download the CHELSA trace21k

Usage

filenames_chelsa_trace21k(dataset, bio_var)

Arguments

dataset

the name of the dataset of interest (currently unused)
bio_var

the variable of interest

Value

a vector of times, one per band
