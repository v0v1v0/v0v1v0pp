Pfr_P_ratio_mono R Documentation

Calculation of Pfr:Ptot ratio for monochromatic radiation.

Description

Ratio between Pr and Ptot (photoequilibrium) for Type I Phytochrome.

Usage

Pfr_P_ratio_mono(w.length)

Arguments

w.length

a single value or a vector of wavelengths (nm)

Value

a single value or a vector of numeric values giving the unitless ratio

Warning!

Exact only for narrow band light sources!
