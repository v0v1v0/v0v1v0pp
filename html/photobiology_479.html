green_leaf.spct R Documentation

Green birch leaf reflectance.

Description

A dataset of spectral reflectance expressed as a fraction of one.

Usage

green_leaf.spct

Format

A reflector_spct object with 226 rows and 2 variables

Details

  • w.length (nm)

  • Rfr (0..1)

References
