customerprofiles_list_identity_resolution_jobs R Documentation

Lists all of the Identity Resolution Jobs in your domain

Description

Lists all of the Identity Resolution Jobs in your domain. The response sorts the list by JobStartTime.

See https://www.paws-r-sdk.com/docs/customerprofiles_list_identity_resolution_jobs/ for full documentation.

Usage

customerprofiles_list_identity_resolution_jobs(
  DomainName,
  NextToken = NULL,
  MaxResults = NULL
)

Arguments

DomainName

[required] The unique name of the domain.
NextToken

The token for the next set of results. Use the value returned in the previous response in the next request to retrieve the next set of results.
MaxResults

The maximum number of results to return per page.
