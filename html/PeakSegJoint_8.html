H3K27ac.TDH.MMM4 R Documentation

Some histone ChIP-seq data

Description

This is used in the package tests.

Usage

data("H3K27ac.TDH.MMM4")

Format

A data frame with 29905 observations on the following 5 variables.

sample.id

a factor

chrom

a character vector

chromStart

a numeric vector

chromEnd

a numeric vector

count

a numeric vector
