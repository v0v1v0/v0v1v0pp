df_dose_evid4 R Documentation

A dose data set to be used with EVID=4.

Description

A dataset containing dosing information.

Usage

df_dose_evid4

Format

A data frame with 418 rows and 10 variables:

ID

Individual

PERIOD

Period

TIMEPOINT

Timepoint

TIME

Time, in hours

AMT

Dose, in mg

TRT

Treatment label

DAY

Day of adminsitration

SEQUENCE

Sequence

TRT2

Treatment

EVID

Evid value
