Description

Plot the std mass from GlobalStd algorithm

Usage

plotstd(list)

Arguments

list

a list from getstd function

See Also

getstd, globalstd

Examples

data(spmeinvivo)
pmd <- getpaired(spmeinvivo)
std <- getstd(pmd)
plotstd(std)
