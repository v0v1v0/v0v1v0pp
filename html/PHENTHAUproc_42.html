remove_false R Documentation

Remove FALSE

Description

Set values in SpatRaster or SpatRaster list to -999 if y is FALSE

Usage

remove_false(x, y)

Arguments

x

SpatRaster/SpatRaster list - numeric - with time attribute
y

SpatRaster - logical - with time attribute
