R2 R Documentation

R-squared

Description

Simple measure of R-squared

Usage

R2(x, y)

Arguments

x

a vector of real numbers
y

a vector of real numbers; must be the same length as x

Value

R2 output is a number between 0 and 1

Author(s)

Joseph G. Kreke, PhD

Examples

 x=c(1,2,3,4,5,6)
 y=c(1.2, 2.1, 2.9, 3.9, 5.3, 6.0)
 r2 <- R2(x,y)
