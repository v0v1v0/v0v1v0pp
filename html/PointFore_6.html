lag R Documentation

Lagging variables for use in estimate functional

Description

Lagging variables for use in estimate functional

Usage

lag(vector, lag = 1)

Arguments

vector

vector to be lagged
lag

number of lags

Value

lagged vector of same length with NAs at beginning

Examples

#lag example vector by one lag
lag(c(1,2,3))

#lag example vector by two lags
lag(c(1,2,3,4),lag=2)
