fis_list_experiment_resolved_targets R Documentation

Lists the resolved targets information of the specified experiment

Description

Lists the resolved targets information of the specified experiment.

See https://www.paws-r-sdk.com/docs/fis_list_experiment_resolved_targets/ for full documentation.

Usage

fis_list_experiment_resolved_targets(
  experimentId,
  maxResults = NULL,
  nextToken = NULL,
  targetName = NULL
)

Arguments

experimentId

[required] The ID of the experiment.
maxResults

The maximum number of results to return with a single call. To retrieve the remaining results, make another call with the returned nextToken value.
nextToken

The token for the next page of results.
targetName

The name of the target.
