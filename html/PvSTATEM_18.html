plot_mfi_for_analyte R Documentation

Plot MFI value distribution for a given analyte

Description

Plot MFI value distribution for a given analyte

Usage

plot_mfi_for_analyte(
  plate,
  analyte_name,
  data_type = "Median",
  plot_type = "boxplot"
)

Arguments

plate

A plate object
analyte_name

The analyte to plot
data_type

The type of data to plot. Default is "Median"
plot_type

The type of plot to generate. Default is "boxplot". Available options are "boxplot" and "violin".

Value

A ggplot object
