cognitoidentityprovider_create_user_pool_domain R Documentation

Creates a new domain for a user pool

Description

Creates a new domain for a user pool.

See https://www.paws-r-sdk.com/docs/cognitoidentityprovider_create_user_pool_domain/ for full documentation.

Usage

cognitoidentityprovider_create_user_pool_domain(
  Domain,
  UserPoolId,
  CustomDomainConfig = NULL
)

Arguments

Domain

[required] The domain string. For custom domains, this is the fully-qualified domain name, such as auth.example.com. For Amazon Cognito prefix domains, this is the prefix alone, such as auth.
UserPoolId

[required] The user pool ID.
CustomDomainConfig

The configuration for a custom domain that hosts the sign-up and sign-in webpages for your application.

Provide this parameter only if you want to use a custom domain for your user pool. Otherwise, you can exclude this parameter and use the Amazon Cognito hosted domain instead.

For more information about the hosted domain and custom domains, see Configuring a User Pool Domain.
