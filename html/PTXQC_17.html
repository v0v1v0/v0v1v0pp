darken R Documentation

Make a color (given as name or in RGB) darker by factor x = [0 = black, 1=unchanged]

Description

Make a color (given as name or in RGB) darker by factor x = [0 = black, 1=unchanged]

Usage

darken(color, factor = 0.8)

Arguments

color

A color as understood by col2rgb
factor

Between 0 (make black) and 1 (leave color as is)

Value

darkened color
