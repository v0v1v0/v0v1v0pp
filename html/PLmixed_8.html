KYPSitemsim R Documentation

Simulated KYPS item-level dataset.

Description

A simulated dataset that replicates the dataset item-level data from KYPS.

Usage

KYPSitemsim

Format

A data frame with 66947 rows and 6 variables:

id

Student ID

time

Time Identifier

item

Item ID

mid

Middle School ID

hid

High School ID

response

Item Response
