phrSetDumpFileOn R Documentation

Set the dump file on/off.

Description

Sets the dump file switch on or off. This switch controls whether or not phreeqc writes to the dump file. The initial setting is off.

Usage

phrSetDumpFileOn(value)

Arguments

value

if TRUE, captures output normally sent to the dump file into a buffer.

References

https://water.usgs.gov/water-resources/software/PHREEQC/IPhreeqc.pdf

See Also

Other Dump: phrGetDumpFileName(), phrGetDumpFileOn(), phrGetDumpStrings(), phrGetDumpStringsOn(), phrSetDumpFileName(), phrSetDumpStringsOn()

Examples


# This example equilibrates pure water with calcite and writes the 
# dump results to file.
phrLoadDatabaseString(phreeqc.dat)
phrSetDumpFileOn(TRUE)
input <- c( 
  'SOLUTION 1 Pure water     ',
  'EQUILIBRIUM_PHASES 1      ',
  '    Calcite 0 10          ',
  'SAVE solution 1           ',
  'SAVE equilibrium_phases 1 ',
  'DUMP                      ',
  '    -solution 1           ',
  '    -equilibrium_phases 1 '
  )

if (is.null(phrRunString(input))) {
  cat(paste("see ", phrGetDumpFileName(), ".\n", sep = ""))
}
