Pfr_Ptot R Documentation

Calculate phytochrome photoequilibrium

Description

A method implemented for objects of different classes.

Usage

Pfr_Ptot(x, ...)

## Default S3 method:
Pfr_Ptot(x, ...)

## S3 method for class 'numeric'
Pfr_Ptot(x, spct.out = length(x) > 20, ...)

## S3 method for class 'source_spct'
Pfr_Ptot(x, ...)

Arguments

x

an R object
...

not used
spct.out

logical Flag indicating if the returned object should be of class response_spct instead of numeric.

Value

If x is numeric, giving wavelengths (nm), a vector of numeric values giving the unitless photon ratio at each wavelength or a generic_spct object with the wavelength values sorted in ascending order and the corresponding Pfr_Ptot values in column s.q.response.

If x is a source_spct object, a single numeric value giving the unitless photon ratio

Methods (by class)

  • Pfr_Ptot(default): Default for generic function

  • Pfr_Ptot(numeric): Specialization for numeric

  • Pfr_Ptot(source_spct): Specialization for source_spct

    Calculate phytochrome photoequilibrium from spectral (photon) irradiance
