applicationinsights_list_workloads R Documentation

Lists the workloads that are configured on a given component

Description

Lists the workloads that are configured on a given component.

See https://www.paws-r-sdk.com/docs/applicationinsights_list_workloads/ for full documentation.

Usage

applicationinsights_list_workloads(
  ResourceGroupName,
  ComponentName,
  MaxResults = NULL,
  NextToken = NULL,
  AccountId = NULL
)

Arguments

ResourceGroupName

[required] The name of the resource group.
ComponentName

[required] The name of the component.
MaxResults

The maximum number of results to return in a single call. To retrieve the remaining results, make another call with the returned NextToken value.
NextToken

The token to request the next page of results.
AccountId

The AWS account ID of the owner of the workload.
