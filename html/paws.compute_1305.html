lightsail_stop_gui_session R Documentation

Terminates a web-based NICE DCV session that’s used to access a virtual computer’s operating system or application

Description

Terminates a web-based NICE DCV session that’s used to access a virtual computer’s operating system or application. The session will close and any unsaved data will be lost.

See https://www.paws-r-sdk.com/docs/lightsail_stop_gui_session/ for full documentation.

Usage

lightsail_stop_gui_session(resourceName)

Arguments

resourceName

[required] The resource name.
