Description

renders the output of the piped functions to the console via cat()

Usage

show(prev = NULL)

Arguments

prev

a placeholder for the output of functions being piped into show(). Defaults to NULL and should not be altered.

Value

None (invisible NULL)

Examples

go("https://www.wikipedia.org/") %>>%
