try_again R Documentation

Try a code snipped multiple times.

Description

Adapted from function try_again from package testthat.

Usage

try_again(
  x,
  times = 2,
  delay_seconds_min = 5,
  delay_seconds_max = 10,
  verbose = FALSE
)

Arguments

x

code
times

Number of times to try
delay_seconds_min

Number of seconds to delay on failure
delay_seconds_max

Number of seconds to delay on failure
verbose

Boolean. Do you want information?
