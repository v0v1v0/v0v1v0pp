tidy.nullmodel R Documentation

Tidy method for null models

Description

Return the results of nullmodel as a tibble

Usage

## S3 method for class 'nullmodel'
tidy(x, ...)

Arguments

x

A nullmodel object.
...

Not used.

Value

A tibble with column value.

Examples



nullmodel(mtcars[,-1], mtcars$mpg) %>% tidy()
