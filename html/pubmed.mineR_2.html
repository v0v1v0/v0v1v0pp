additional_info R Documentation

To extract sentences with nultiple keywords from Abstracts

Description

additional_info will help to extract the sentences containing multiple query term(s) from a large corpus of multiple abstracts.

Usage

additional_info(abs, pmid, keywords)

Arguments

abs

abs an S4 object of class Abstracts.

pmid

Vector of PMIDs from abstracts

keywords

Character Vector of Terms

Value

It will return a matrix object containing PMID, keywords and sentences
