distance R Documentation

Function to calculate distance between two vectors

Description

Function to calculate distance between two vectors

Usage

distance(x, y, method = "cosine")

Arguments

x

numeric vector
y

numeric vector
method

distance calculation method: cosine (default), dot.prod, euclidian, manhattan, chebyshev, coassociation

Value

the distance value

Examples

x = c(1,2,3)
y = c(6,4,6)
distance(x,y)
