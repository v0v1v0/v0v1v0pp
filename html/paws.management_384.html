cloudwatchevents_list_partner_event_sources R Documentation

An SaaS partner can use this operation to list all the partner event source names that they have created

Description

An SaaS partner can use this operation to list all the partner event source names that they have created. This operation is not used by Amazon Web Services customers.

See https://www.paws-r-sdk.com/docs/cloudwatchevents_list_partner_event_sources/ for full documentation.

Usage

cloudwatchevents_list_partner_event_sources(
  NamePrefix,
  NextToken = NULL,
  Limit = NULL
)

Arguments

NamePrefix

[required] If you specify this, the results are limited to only those partner event sources that start with the string you specify.
NextToken

The token returned by a previous call to this operation. Specifying this retrieves the next set of results.
Limit

pecifying this limits the number of results returned by this operation. The operation also returns a NextToken which you can use in a subsequent operation to retrieve the next set of results.
