getAbstracts R Documentation

get Abstracts

Description

retrieve abstracts of the specified PMIDs from PubMed.

Usage

getAbstracts(pmid, https = TRUE, s = 100)

Arguments

pmid

a set of PMIDs
https

use https instead of http
s

download how many PMIDs each time

See Also

getPMIDs

Examples
