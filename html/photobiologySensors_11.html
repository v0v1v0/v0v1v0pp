deltat_sensors R Documentation

Delta-T sensors

Description

A vector of names useful for extracting subsets of spectra from the sensors.mspct object.

Usage

deltat_sensors

Format

A character vector of names of members of the collection of spectra.

Note

Spectral data digitized from figure in manufacturer's manual. These are typical measured data, and for specific sensor units the calibration data supplied with the sensor must be used as individual units may differ in their spectral response. Furthermore, the digigitizing process from figures with a linear scale is prone to errors, which are specially noticeable in regions of low responsivity.

Manufacturer: Delta-T Devices Ltd, CAMBRIDGE CB25 0EJ, UK https://delta-t.co.uk/
