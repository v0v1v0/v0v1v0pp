finspace_update_environment R Documentation

Update your FinSpace environment

Description

Update your FinSpace environment.

See https://www.paws-r-sdk.com/docs/finspace_update_environment/ for full documentation.

Usage

finspace_update_environment(
  environmentId,
  name = NULL,
  description = NULL,
  federationMode = NULL,
  federationParameters = NULL
)

Arguments

environmentId

[required] The identifier of the FinSpace environment.
name

The name of the environment.
description

The description of the environment.
federationMode

Authentication mode for the environment.

  • FEDERATED - Users access FinSpace through Single Sign On (SSO) via your Identity provider.

  • LOCAL - Users access FinSpace via email and password managed within the FinSpace environment.

federationParameters
