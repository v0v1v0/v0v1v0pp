p_cran R Documentation

CRAN Packages

Description

p_cran - Generate a vector of all available packages.

p_iscran - Logical check if a package is available on CRAN.

Usage

p_cran(menu = FALSE)

p_iscran(package)

Arguments

menu

logical. If TRUE allows user to select the package and return that package name.
package

Name of package.

See Also

available.packages

Examples

## Not run: 
p_cran()
p_cran(TRUE)
p_iscran(pacman)

## End(Not run)
