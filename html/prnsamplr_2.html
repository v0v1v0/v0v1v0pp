ExampleData R Documentation

ExampleData

Description

Artificial dataset to be used with samp and transformprn.

Usage

data("ExampleData")

Format

A data frame with 40000 observations on the following 6 variables.

stratum

a character vector

id

a numeric vector

npopul

a numeric vector

nsample

a numeric vector

rands

a numeric vector

sizeM

a numeric vector

Examples

data(ExampleData)
