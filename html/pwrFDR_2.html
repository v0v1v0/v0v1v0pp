Division operator with divide by zero clobbering

Description

x %over% y = x/y when y!=0, equals 0 when y==0.

Usage

  x %over% y

Arguments

x,y

Numeric or complex vectors or objects that can be coerced to such.

Value

x/y when y!=0, otherwise 0.
