C_T values from qPCR (Serial dilutions)

Description

A dataset containing the C_T values of two genes from a serial dilution qPCR experiment. The original dataset shows only the averages and standard deviations of each of the 7 different dilutions (1, .5, .2, .1, .05, .02 and .01). These summaries were used to regenerate 3 replicates for each of the dilutions to be used in testing and examples of the different functions.

Usage

ct3

Format

A data.frame with 21 rows and 2 variables:

c_myc

C_T values of the target gene c-myc

GAPDH

C_T values of the control gene GAPDH
