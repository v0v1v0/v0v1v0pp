demo.profiles R Documentation

Demo profiles

Description

These profiles resulted in negative means in a buggy version of the PeakSegJointFaster code.

Usage

data("demo.profiles")

Format

A data frame with 51204 observations on the following 5 variables.

sample.id

a character vector

sample.group

a character vector

chromStart

a numeric vector

chromEnd

a numeric vector

count

a numeric vector
