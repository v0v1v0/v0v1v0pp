bank_income R Documentation

Panel data of bank,2001Q1~2010Q1

Description

A quarterly panel data frame with 1000 observations on the following 7 variables, unbalanced panel data ranges from 2001Q1~2010Q1.

Usage

data("bank_income")

Format

ID

a numeric vector

Qtr

a numeric vector

preTax_Income

a numeric vector

shortRatio

a numeric vector

longRatio

a numeric vector

Current_ratio

a numeric vector

LoanDeposit_ratio

a numeric vector

Examples

data(bank_income)
