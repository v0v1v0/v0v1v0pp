sugar R Documentation

A real data set on sold sugar

Description

A collection of scanner data on the sale of sugar in one of Polish supermarkets in the period from December 2017 to October 2020

Usage

sugar

Format

A data frame with 6 columns and 7666 rows. The used variables are as follows:

time - Dates of transactions (Year-Month-Day)

prices - Prices of sold products [PLN]

quantities - Quantities of sold products [kg]

prodID - Unique product codes (data set contains 11 different prodIDs)

retID - Unique codes identifying outlets/retailer sale points (data set contains 20 different retIDs)

description Descriptions of sold sugar products (data set contains 3 different product descriptions)
