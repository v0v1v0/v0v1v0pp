prereg_pdf R Documentation

Preregistration renderer

Description

Knit a PDF document using preregistration document template

Usage

prereg_pdf(...)

aspredicted_prereg(...)

brandt_prereg(...)

cos_prereg(...)

fmri_prereg(...)

psyquant_prereg(...)

prp_quant_prereg(...)

rr_prereg(...)

vantveer_prereg(...)

Arguments

...

additional arguments to pdf_document; template is ignored.
