Polished API - Get App(s) User(s)

Description

Polished API - Get App(s) User(s)

Usage

get_app_users(
  app_uid = NULL,
  user_uid = NULL,
  email = NULL,
  api_key = get_api_key()
)

Arguments

app_uid

an optional app uid.
user_uid

an optional user uid.
email

an optional user email address.
api_key

your Polished API key. Set your polished api key using set_api_key() so that you do not need to supply this argument with each function call.

Details

If app_uid, user_uid, & email are all NULL, then all app users will be returned.

Value

an object of class polished_api_res. When successful, the content of the object is a tibble of app(s) with the following columns:

  • uid

  • app_uid

  • user_uid

  • is_admin

  • created_at

  • email
