bedrock_stop_evaluation_job R Documentation

Stops an in progress model evaluation job

Description

Stops an in progress model evaluation job.

See https://www.paws-r-sdk.com/docs/bedrock_stop_evaluation_job/ for full documentation.

Usage

bedrock_stop_evaluation_job(jobIdentifier)

Arguments

jobIdentifier

[required] The ARN of the model evaluation job you want to stop.
