sex R Documentation

Sexual Activity in Risk Behavior Survey

Description

Information on sexual activities from the (RBS)

Usage

data(sex)

Format

A tibble with 3,299 rows and 40 variables:

who
Type: integer
Description: Patient ID
txx_prt
Type: factor (First/Reference level = 0)
Description: Total sex partners:
  • 0 = 0
  • 1 = 1
  • 2 = more than one
Levels: 0, 1, 2
txx_mprt
Type: factor (First/Reference level = 0)

  • 1 = 1
  • 2 = more than one
