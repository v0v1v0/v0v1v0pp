get_type_RENDIS R Documentation

Retrieve information about soil defense type of the ReNDiS database

Description

The get_type_RENDIS function returns the list of type of interventions for soil defense contained in the ReNDiS database.

Usage

get_type_RENDIS()

Value

Return an object of class data.frame

Types:

  • Frana => Landslide

  • Non definito => Not defined

  • Alluvione => Flooding

  • Misto => Mixed

  • Valanga => Avalanche

  • Incendio => Wildfire

  • Costiero => Coastal
