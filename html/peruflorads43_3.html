ds43_2006_sps_class R Documentation

List of species name in tab_ds43_2006 separeted by category

Description

The 'ds43_2006_sps_class' includes all species separeted by genus, epithet, author, subspecies, variety, and id (position in the tab_ds43_2006).

Usage

ds43_2006_sps_class

Format

A data.frame.

Examples


data(ds43_2006_sps_class)
