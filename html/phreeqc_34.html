llnl.dat R Documentation

The llnl.dat database.

Description

llnl.dat is a database derived from databases for EQ3/6 and Geochemist's Workbench that uses thermodynamic data compiled by the Lawrence Livermore National Laboratory. The database has been reformatted for use by phrLoadDatabaseString.

Usage

llnl.dat  # phrLoadDatabaseString(llnl.dat)

References

https://pubs.usgs.gov/tm/06/a43/pdf/tm6-A43.pdf
