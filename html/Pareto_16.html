is.PPP_Model R Documentation

Check if an object is a PPP_Model

Description

Checks if the class of an object is 'PPP_Model'

Usage

is.PPP_Model(x)

Arguments

x

Object to be checked.

Examples

PPPM <- PPP_Model(2, c(1000,2000), c(1,2), dispersion = 2)
PPPM
is.valid.PPP_Model(PPPM)

PPPM$alpha <- 2
is.valid.PPP_Model(PPPM)
is.PPP_Model(PPPM)
