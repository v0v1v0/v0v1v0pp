vpclattice_delete_rule R Documentation

Deletes a listener rule

Description

Deletes a listener rule. Each listener has a default rule for checking connection requests, but you can define additional rules. Each rule consists of a priority, one or more actions, and one or more conditions. You can delete additional listener rules, but you cannot delete the default rule.

See https://www.paws-r-sdk.com/docs/vpclattice_delete_rule/ for full documentation.

Usage

vpclattice_delete_rule(listenerIdentifier, ruleIdentifier, serviceIdentifier)

Arguments

listenerIdentifier

[required] The ID or Amazon Resource Name (ARN) of the listener.
ruleIdentifier

[required] The ID or Amazon Resource Name (ARN) of the rule.
serviceIdentifier

[required] The ID or Amazon Resource Name (ARN) of the service.
