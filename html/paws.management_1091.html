prometheusservice_tag_resource R Documentation

The TagResource operation associates tags with an Amazon Managed Service for Prometheus resource

Description

The tag_resource operation associates tags with an Amazon Managed Service for Prometheus resource. The only resources that can be tagged are workspaces and rule groups namespaces.

See https://www.paws-r-sdk.com/docs/prometheusservice_tag_resource/ for full documentation.

Usage

prometheusservice_tag_resource(resourceArn, tags)

Arguments

resourceArn

[required] The ARN of the workspace or rule groups namespace to apply tags to.
tags

[required] The list of tag keys and values to associate with the resource.

Keys may not begin with ⁠aws:⁠.
