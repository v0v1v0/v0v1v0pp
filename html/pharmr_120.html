has_linear_odes_with_real_eigenvalues

Description

Check if model has a linear ode system with real eigenvalues

Usage

has_linear_odes_with_real_eigenvalues(model)

Arguments

model

(Model) Pharmpy model

Value

(logical) TRUE if model has an ODE system that is linear

has_linear_odes
