date2doy R Documentation

Date to Julian Day Converter

Description

Converts a date into a Julian day

Usage

date2doy(date)

Arguments

date

Date YYMMDD as integer

Value

The Julian Day (day of year) of the date

Author(s)

Daniel Doktor, Maximilian Lange

Examples

	date <- 891208
	doy <- 0
	doy <- date2doy(date)
	doy
