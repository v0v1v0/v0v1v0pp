bioconductorRank R Documentation

Package download counts and rank percentiles.

Description

From bioconductor

Usage

bioconductorRank(packages = "monocle", date = "2019-01",
  count = "download")

Arguments

packages

Character. Vector of package name(s).
date

Character. Date. yyyy-mm
count

Character. "ip" or "download".

Value

An R data frame.

Examples

bioconductorRank(packages = "cicero", date = "2019-09")
