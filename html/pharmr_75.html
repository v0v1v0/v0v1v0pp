get_baselines R Documentation

get_baselines

Description

Baselines for each subject.

Baseline is taken to be the first row even if that has a missing value.

Usage

get_baselines(model)

Arguments

model

(Model) Pharmpy model

Value

(data.frame) Dataset with the baselines

Examples

## Not run: 
model <- load_example_model("pheno")
get_baselines(model)

## End(Not run)
