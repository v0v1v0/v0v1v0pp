Amm.dat R Documentation

The Amm.dat database.

Description

Amm.dat is the same as phreeqc.dat, except that ammonia redox state has been decoupled from the rest of the nitrogen system; that is, ammonia has been defined as a separate component. The database has been reformatted for use by phrLoadDatabaseString.

Usage

Amm.dat  # phrLoadDatabaseString(Amm.dat)

Source

http://wwwbrr.cr.usgs.gov/projects/GWC_coupled/phreeqc
