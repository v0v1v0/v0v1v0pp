repEach R Documentation

Repeat each element x_i in X, n_i times.

Description

Repeat each element x_i in X, n_i times.

Usage

repEach(x, n)

Arguments

x

Values to be repeated
n

Number of repeat for each x_i (same length as x)

Value

Vector with values from x, n times

Examples


  repEach(1:3, 1:3)  ## 1, 2, 2, 3, 3, 3
