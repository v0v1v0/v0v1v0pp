BulletList R Documentation

Bullet List

Description

Constructs a block object of type "BulletList".

Usage

BulletList(llblocks)

Arguments

llblocks

a list of lists of blocks

Examples

bullet_1 <- Plain("A")
bullet_2 <- Plain(Str("B"))
bullet_3 <- list(Plain(list(Str("C"))))
BulletList(list(bullet_1, bullet_2, bullet_3))
