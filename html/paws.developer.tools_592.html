Usage

wellarchitected_list_notifications(
  WorkloadId = NULL,
  NextToken = NULL,
  MaxResults = NULL,
  ResourceArn = NULL

Arguments

WorkloadId
NextToken
MaxResults

ResourceArn

The ARN for the related resource for the notification.
