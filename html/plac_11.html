TvInd R Documentation

Generate time-depependent covariate indicators

Description

Generate time-depependent covariate indicators

Usage

TvInd(zeta, W)

Arguments

zeta

the change point (jump time) of Z_v(t).
W

the ordered observed event times.

Value

the time-depependent covariate of the form Z_v(t) = I(w_k > zeta).
