elasticsearchservice_update_package R Documentation

Updates a package for use with Amazon ES domains

Description

Updates a package for use with Amazon ES domains.

See https://www.paws-r-sdk.com/docs/elasticsearchservice_update_package/ for full documentation.

Usage

elasticsearchservice_update_package(
  PackageID,
  PackageSource,
  PackageDescription = NULL,
  CommitMessage = NULL
)

Arguments

PackageID

[required] Unique identifier for the package.
PackageSource

[required]
PackageDescription

New description of the package.
CommitMessage

An info message for the new version which will be shown as part of GetPackageVersionHistoryResponse.
