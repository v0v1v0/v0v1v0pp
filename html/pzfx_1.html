Description

List all tables in a 'GraphPad Prism' '.pzfx' file.

Usage

pzfx_tables(path)

Arguments

path

Path to the '.pzfx' file

Value

a character string vector

Examples

pzfx_file <- system.file("extdata/exponential_decay.pzfx", package = "pzfx", mustWork = TRUE)
pzfx_tables(pzfx_file)
