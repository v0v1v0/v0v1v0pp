is.standardized R Documentation

Check if a dataframe is standardized.

Description

Check if a dataframe is standardized.

Usage

is.standardized(df, tol = 0.1)

Arguments

df

A dataframe.
tol

The error treshold.

Value

bool.

Author(s)

Dominique Makowski

Examples

library(psycho)
library(effectsize)

df <- psycho::affective
is.standardized(df)

dfZ <- effectsize::standardize(df)
is.standardized(dfZ)
