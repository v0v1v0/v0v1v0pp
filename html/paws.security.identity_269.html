cognitoidentityprovider_list_identity_providers R Documentation

Usage

cognitoidentityprovider_list_identity_providers(
  UserPoolId,
  MaxResults = NULL,
  NextToken = NULL
)

Arguments

UserPoolId

[required] The user pool ID.
MaxResults

The maximum number of IdPs to return.
NextToken
