paymentcryptographycontrolplane_delete_alias R Documentation

Deletes the alias, but doesn't affect the underlying key

Description

Deletes the alias, but doesn't affect the underlying key.

See https://www.paws-r-sdk.com/docs/paymentcryptographycontrolplane_delete_alias/ for full documentation.

Usage

paymentcryptographycontrolplane_delete_alias(AliasName)

Arguments

AliasName

[required] A friendly name that you can use to refer Amazon Web Services Payment Cryptography key. This value must begin with ⁠alias/⁠ followed by a name, such as alias/ExampleAlias.
