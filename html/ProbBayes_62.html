pythag2018 R Documentation

Baseball Win-Loss Records

Description

Final standings of the MLB baseball teams in the 2018 season

Usage

  pythag2018

Format

A data frame with 30 observations on the following 7 variables.

Team

team abbreviation

League

league abbreviation

W

number of wins

L

number of losses

Pct

proportion of wins

R

average runs scored

RA

average runs allowed

Source

Lahman database
