C_T values from qPCR (Serial dilutions)

Description

A dataset containing the C_T values of two genes from a controlled serial dilution qPCR experiment. The data were prepared from four different dilutions (10, 2, 0.4 and 0.08) and two control groups; control and treatment (n = 12) each.

Usage

ct4

Format

A data.frame with 24 rows and 2 variables:

ref

C_T values of the reference gene

target

C_T values of the target gene

Source

https://www.ncbi.nlm.nih.gov/pmc/articles/PMC1395339/
