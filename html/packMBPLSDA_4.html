disjunctive R Documentation

Disjunctive table

Description

Function to transform a boolean matrix in a disjunctive table

Usage

disjunctive(y)

Arguments

y

boolean matrix indicating observations categories

Details

no details are needed

Value

ydisj

disjunctive table

Author(s)
