cognitoidentity_list_identities R Documentation

Lists the identities in an identity pool

Description

Lists the identities in an identity pool.

See https://www.paws-r-sdk.com/docs/cognitoidentity_list_identities/ for full documentation.

Usage

cognitoidentity_list_identities(
  IdentityPoolId,
  MaxResults,
  NextToken = NULL,
  HideDisabled = NULL
)

Arguments

IdentityPoolId

[required] An identity pool ID in the format REGION:GUID.
MaxResults

[required] The maximum number of identities to return.
NextToken

A pagination token.
HideDisabled

An optional boolean parameter that allows you to hide disabled identities. If omitted, the ListIdentities API will include disabled identities in the response.
