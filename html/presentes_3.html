parque_de_la_memoria R Documentation

parque_de_la_memoria

Description

Dataset elaborado a partir de los registros de http://basededatos.parquedelamemoria.org.ar/registros/

Usage

parque_de_la_memoria

Format

Data frame con 8751 filas y 30 variables:

id

character id del caso. Construido a partir del id utilizado en la página

character
