frezchem.dat R Documentation

The frezchem.dat database

Description

frezchem.dat is a database from Jonathon Toner and the developers for the frezchem code (Toner and Sletten (2013), Marion and coworkers). Frezchem is a Pitzer model for temperatures of 0 C and below. phrLoadDatabaseString.

Usage

frezchem.dat  # phrLoadDatabaseString(frezchem.dat)
