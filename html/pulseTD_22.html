splitB4Params R Documentation

esitmate degradation parameters

Description

It is used to solve the degradation parameters, including the error function and the optimization function.

Usage

splitB4Params(TT, pre_a, TimeGrid)

Arguments

TT

exon expression value of total RNA
pre_a

Parameter vector of transcription pulse function
TimeGrid

a vector of times

Value

a vector of degradation parameters,contains pulse parameters and fitting error values
