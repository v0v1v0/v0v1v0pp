create_grob R Documentation

Create Heatmap Graphical Object

Description

Combines the tree, heatmap, and titles to create final heatmap image.

Usage

create_grob(report, grob.title = "Plasmid Profiles")

Arguments

report

Dataframe of results
grob.title

Title of heatmap

Value

Composite image

Examples

## Not run: 
create_grob(report, grob.title="Plasmid Profiles")

## End(Not run)
