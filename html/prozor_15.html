make id for chain in format sp|P30443|1A01_HUMANs25

Description

make id for chain in format sp|P30443|1A01_HUMANs25

Usage

makeID(sequence, id, sp)

Arguments

sequence

- aa sequence as string
id

uniprot id id: sp|P30443|1A01_HUMAN
sp

start position of chain numeric

Value

string consisting of id,"s",sp
