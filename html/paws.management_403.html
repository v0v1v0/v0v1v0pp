cloudwatchevents_update_archive R Documentation

Updates the specified archive

Description

Updates the specified archive.

See https://www.paws-r-sdk.com/docs/cloudwatchevents_update_archive/ for full documentation.

Usage

cloudwatchevents_update_archive(
  ArchiveName,
  Description = NULL,
  EventPattern = NULL,
  RetentionDays = NULL
)

Arguments

ArchiveName

[required] The name of the archive to update.
Description

The description for the archive.
EventPattern

The event pattern to use to filter events sent to the archive.
RetentionDays

The number of days to retain events in the archive.
