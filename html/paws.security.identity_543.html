guardduty_list_publishing_destinations R Documentation

Returns a list of publishing destinations associated with the specified detectorId

Description

Returns a list of publishing destinations associated with the specified detectorId.

See https://www.paws-r-sdk.com/docs/guardduty_list_publishing_destinations/ for full documentation.

Usage

guardduty_list_publishing_destinations(
  DetectorId,
  MaxResults = NULL,
  NextToken = NULL
)

Arguments

DetectorId

[required] The ID of the detector to retrieve publishing destinations for.
MaxResults

The maximum number of results to return in the response.
NextToken

A token to use for paginating results that are returned in the response. Set the value of this parameter to null for the first request to a list action. For subsequent calls, use the NextToken value returned from the previous request to continue listing results after the first page.
