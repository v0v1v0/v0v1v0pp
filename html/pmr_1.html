Probability Models for Ranking Data

Description

Probability models for ranking data.

Details

Package: pmr
Type: Package
Version: 1.2.5
Date: 2015-05-14
Depends: stats4, graphics
License: GPL-2

This package includes various probability models for ranking data. Current, only complete rankings are supported by this package.
