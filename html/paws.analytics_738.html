ivsrealtime_tag_resource R Documentation

Adds or updates tags for the AWS resource with the specified ARN

Description

Adds or updates tags for the AWS resource with the specified ARN.

See https://www.paws-r-sdk.com/docs/ivsrealtime_tag_resource/ for full documentation.

Usage

ivsrealtime_tag_resource(resourceArn, tags)

Arguments

resourceArn

[required] The ARN of the resource to be tagged. The ARN must be URL-encoded.
tags

[required] Array of tags to be added or updated. Array of maps, each of the form string:string (key:value). See Tagging AWS Resources for details, including restrictions that apply to tags and "Tag naming limits and requirements"; Amazon IVS has no constraints beyond what is documented there.
