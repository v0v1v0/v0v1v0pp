elapsedtime R Documentation

Utility function

Description

Gadget to compute the elapsed time of a process

Usage

elapsedtime(st, et)

Arguments

st

start time
et

end time

Details

Start and end times can be obtained with proc.time.

Value

String with the elapsed time in hh:mm:ss format.
