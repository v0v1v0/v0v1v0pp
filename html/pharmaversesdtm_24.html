rs_onco_imwg R Documentation

Disease Response (IMWG)

Description

A SDTM RS dataset using IMWG criteria. The dataset contains just a few patients. It is intended for vignettes and examples of ADaM dataset creation.

Usage

rs_onco_imwg

Format

An object of class tbl_df (inherits from tbl, data.frame) with 65 rows and 17 columns.
