vecToArrayStr R Documentation

Write a PostgreSQL array as a string using ARRAY[] format from a vector.

Description

Write a PostgreSQL array as a string using ARRAY[] format from a vector.

Usage

vecToArrayStr(x, quote = TRUE)

Arguments

x

A vector.
quote

TRUE/FALSE, if TRUE, the elements of x will be quoted.

Value

A string.

Examples

vecToArrayStr(c("a", "b"))
