p_news R Documentation

Package/R News

Description

Find out news on a package or R.

Usage

p_news(package = NULL)

Arguments

package

Name of package (default is to see news for R).

See Also

news

Examples

## Not run: 
p_news()
p_news(lattice)
## Grab specific version subsets
subset(p_news(lattice), Version == 0.7)  

## End(Not run)
