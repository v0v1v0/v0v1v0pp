fac_to_vect R Documentation

Vectorize Fac object

Description

Vectorize Fac object

Usage

fac_to_vect(Fac)

Arguments

Fac

Fac object output of parafac.

Value

Vectorized Fac object

Examples

set.seed(123)
A = array(rnorm(108*2), c(108, 2))
B = array(rnorm(100*2), c(100, 2))
C = array(rnorm(10*2), c(10, 2))
Fac = list(A, B, C)
v = fac_to_vect(Fac)
