workdocs_update_document_version R Documentation

Changes the status of the document version to ACTIVE

Description

Changes the status of the document version to ACTIVE.

See https://www.paws-r-sdk.com/docs/workdocs_update_document_version/ for full documentation.

Usage

workdocs_update_document_version(
  AuthenticationToken = NULL,
  DocumentId,
  VersionId,
  VersionStatus = NULL
)

Arguments

AuthenticationToken

Amazon WorkDocs authentication token. Not required when using Amazon Web Services administrator credentials to access the API.
DocumentId

[required] The ID of the document.
VersionId

[required] The version ID of the document.
VersionStatus

The status of the version.
