BinaryVectorCheck R Documentation

Checks if a vector is binary

Description

Checks if all the entries of a vector are 0 or 1.

Usage

BinaryVectorCheck(x)

Arguments

x

The vector to check

Value

The logical result.

Author(s)

Jose Luis Vicente-Villardon

Examples

x=c(0, 0, 0, 0,  1, 1, 1, 2)
BinaryVectorCheck(x)
