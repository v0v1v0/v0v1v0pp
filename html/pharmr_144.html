plot_cwres_vs_idv

Description

Plot CWRES vs idv

Usage

plot_cwres_vs_idv(model, residuals, stratify_on = NULL, bins = 8)

Arguments

model

(Model) Pharmpy model
residuals

(data.frame) DataFrame containing CWRES
stratify_on

(str) Name of parameter for stratification
bins

(numeric) Number of bins for stratification

Value

(alt.Chart) Plot
