%relies% R Documentation

Check relies

Description

Check whether a package has a recursive dependency on the other

Usage

pkg_1 %relies% pkg_2

Arguments

pkg_1

(string) A package name
pkg_2

(string) A package name

Value

(flag) TRUE, if 'pkg_1' ‘relies' on ’pkg_2'

Author(s)

Srikanth KS

See Also

relies, reverse_relies

Examples

pkggraph::init(local = TRUE)
"dplyr" %relies% "tibble"
