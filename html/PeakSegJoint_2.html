chr7.peaks R Documentation

chr7 peaks

Description

peaks predicted by PeakSegFPOP on chr7:50410631-53200000 in the H3K36me3_TDH_immune data set.

Usage

data("chr7.peaks")

Format

A data frame with 41 observations on the following 3 variables.

sample.id

a character vector

chromStart

a numeric vector

chromEnd

a numeric vector
