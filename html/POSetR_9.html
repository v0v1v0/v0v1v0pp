incidenceMatrix R Documentation

Poset incidence matrix

Description

The function returns the incidence matrix for objects of class poset.

Usage

incidenceMatrix(x)

Arguments

x

an object of class poset.

Value

a logical square matrix.

Examples

dom <- matrix(c(
  "a", "b",
  "c", "b",
  "b", "d"
), ncol = 2, byrow = TRUE)
p <- poset(x = dom)
incidenceMatrix(p)
