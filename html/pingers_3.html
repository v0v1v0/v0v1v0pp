pingers_heatmap R Documentation

Packet Loss Heatmap

Description

Generates a heatmap that displays the packet loss hotspots on an hourly basis during the week.

Usage

pingers_heatmap(logs = NULL)

Arguments

logs

network_logs file

Value

highcharts heatmap

Examples

## Not run: 
pingers_heatmap(net_logs)

## End(Not run)
