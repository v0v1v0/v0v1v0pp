PowerLogistic R Documentation

The Power Logistic Distribution

Description

Density, distribution function, quantile function and random generation for the power logistic distribution with parameters mu, sigma and lambda.

Usage

dplogis(x, lambda = 1, mu = 0, sigma = 1, log = FALSE)

pplogis(q, lambda = 1, mu = 0, sigma = 1, lower.tail = TRUE,
  log.p = FALSE)

qplogis(p, lambda = 1, mu = 0, sigma = 1, lower.tail = TRUE,
  log.p = FALSE)

rplogis(n, lambda = 1, mu = 0, sigma = 1)

Arguments

x, q

vector of quantiles.
lambda

shape parameter.
mu, sigma

location and scale parameters.
log, log.p

logical; if TRUE, probabilities p are given as log(p).
lower.tail

logical; if TRUE (default), probabilities are P[X \le x ], otherwise, P[X > x].
p

vector of probabilities.
n

number of observations.

Details

The power Logistic distribution has density

f(x)=\lambda \left [\frac{1}{1+e^{-\left ( \frac{x-\mu}{\sigma} \right )} }\right ]^{\lambda-1}\left[\frac{ e^{-\left ( \frac{x-\mu}{\sigma} \right )} }{\sigma\left ( 1+e^{-\left ( \frac{x-\mu}{\sigma} \right )} \right )^2}\right], where -\infty<\mu<\infty is the location paramether, \sigma^2>0 the scale parameter and \lambda>0 the shape parameter.
