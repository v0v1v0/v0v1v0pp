sagemaker_create_artifact R Documentation

Creates an artifact

Description

Creates an artifact. An artifact is a lineage tracking entity that represents a URI addressable object or data. Some examples are the S3 URI of a dataset and the ECR registry path of an image. For more information, see Amazon SageMaker ML Lineage Tracking.

See https://www.paws-r-sdk.com/docs/sagemaker_create_artifact/ for full documentation.

Usage

sagemaker_create_artifact(
  ArtifactName = NULL,
  Source,
  ArtifactType,
  Properties = NULL,
  MetadataProperties = NULL,
  Tags = NULL
)

Arguments

ArtifactName

The name of the artifact. Must be unique to your account in an Amazon Web Services Region.
Source

[required] The ID, ID type, and URI of the source.
ArtifactType

[required] The artifact type.
Properties

A list of properties to add to the artifact.
MetadataProperties
Tags

A list of tags to apply to the artifact.
