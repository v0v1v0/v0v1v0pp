vitaltech_sensors R Documentation

Vital Technologies sensors

Description

A vector of names useful for extracting subsets of spectra from the sensors.mspct object.

Usage

vitaltech_sensors

Format

A character vector of names of members of the collection of spectra.

Details

Vital "Blue Wave" BW-20 UV-B "erythemal" radiometer. Data for a specific unit included in a sensor intercomparison event.

Manufacturer: Vital Technologies, Canada. Company no longer in business.

Source

Data kindly made available by Lasse Ylianttila.

References

Leszczynski K, Jokela K, Ylianttila L, Visuri R, Blumthaler M. 1997. Report of the WMO/STUK Intercomparison of erythemally-weighted solar UV radiometers (Spring/Summer 1995, Helsinki, Finland). WMO-GAW Report No. 112, 90 pages.

Leszczynski K, Jokela K, Ylianttila L, Visuri R, Blumthaler M. 1998. Erythemally weighted radiometers in solar UV monitoring: results from the WMO/STUK Intercomparison. Photochem. Photobiol. 67(2):212-221.
