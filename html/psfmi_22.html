ipdna_md R Documentation

Example dataset for the psfmi_mm function

Description

5 imputed datasets of the first 10 centres of the IPDNa dataset in the micemd package.

Usage

data(ipdna_md)

Format

A data frame with 13390 observations on the following 13 variables.

.imp

a numeric vector

.id

a numeric vector

centre

cluster variable

gender

dichotomous

bmi

continuous

age

continuous

sbp

continuous

dbp

continuous

hr

continuous

lvef

dichotomous

bnp

categorical

afib

continuous

bmi_cat

categorical

Examples

data(ipdna_md)
