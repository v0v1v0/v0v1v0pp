pharmaverseadam: ADaM Test Data for the 'Pharmaverse' Family of Packages

Description

A set of Analysis Data Model (ADaM) datasets constructed using the Study Data Tabulation Model (SDTM) datasets contained in the 'pharmaversesdtm' package and the template scripts from the 'admiral' family of packages. ADaM dataset specifications are described in: CDISC Analysis Data Model Team (2021) https://www.cdisc.org/system/files/members/standard/foundational/ADaMIG_v1.3.pdf.
