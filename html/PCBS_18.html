Ol_Reliable R Documentation

PCBS ggplot theme.

Description

Custom theme for ggplot used by all PCBS output figures.

Usage

Ol_Reliable()

Value

Theme for ggplot objects used by PCBS.

Examples

df <- data.frame(A=c(1,2,3), B=c(1,2,3))

ggplot2::ggplot(df, ggplot2::aes(x=A, y=B))+
ggplot2::geom_point()+
Ol_Reliable()
