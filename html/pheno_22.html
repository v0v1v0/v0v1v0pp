tau R Documentation

Kendall's normalized tau

Description

Kendall's normalized tau for time series x

Usage

tau(x)

Arguments

x

Numeric vector x.

Details

Implicitly assumes a equidistant time series x.

Value

t

Kendall's normalized tau.

Author(s)

Joerg Schaber
