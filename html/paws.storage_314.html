omics_list_reference_import_jobs R Documentation

Retrieves a list of reference import jobs

Description

Retrieves a list of reference import jobs.

See https://www.paws-r-sdk.com/docs/omics_list_reference_import_jobs/ for full documentation.

Usage

omics_list_reference_import_jobs(
  maxResults = NULL,
  nextToken = NULL,
  referenceStoreId,
  filter = NULL
)

Arguments

maxResults

The maximum number of jobs to return in one page of results.
nextToken

Specify the pagination token from a previous request to retrieve the next page of results.
referenceStoreId

[required] The job's reference store ID.
filter

A filter to apply to the list.
