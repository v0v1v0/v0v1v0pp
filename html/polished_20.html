email_input R Documentation

A Shiny email input

Description

This is a replica of shiny::textInput() with the HTML input type attribute set to "email" rather than "text".

Usage

email_input(
  inputId,
  label = tagList(shiny::icon("envelope"), "Email"),
  value = "",
  width = NULL,
  placeholder = NULL
)

Arguments

inputId

The input slot that will be used to access the value.
label

Display label for the control, or NULL for no label.
value

Initial value.
width

The width of the input, e.g. '400px'.
placeholder

A character string giving the user a hint as to what can be entered into the control. Internet Explorer 8 and 9 do not support this option.

Value

the UI for the email input.
