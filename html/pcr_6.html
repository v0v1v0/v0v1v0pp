.pcr_normalize R Documentation

Normalize vector by another

Description

Normalize vector by another

Usage

.pcr_normalize(vec, ref, mode = "subtract")

Arguments

vec

A vector of numerics
ref

A numeric vector
mode

Either 'subtract' or 'divide'

Value

A vector of numerics

Examples

vec <- rnorm(6, 30, 1)
ref <- rnorm(6, 30, .1)
pcr:::.pcr_normalize(vec, ref)
pcr:::.pcr_normalize(vec, ref, mode = 'divide')
