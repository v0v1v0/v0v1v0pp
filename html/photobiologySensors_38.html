thiesclima_sensors R Documentation

Thies Clima sensors

Description

A vector of names useful for extracting subsets of spectra from the sensors.mspct object.

Usage

thiesclima_sensors

Format

A character vector of names of members of the collection of spectra.

Details

Thies Clima E1.c broadband UVB sensor

Source

Digitized with 'DigitizeIt' from manufacturers manual. These are approximate data, both because of the digitizing process, and because they are either typical values or for a particular sensor unit. Individual sensor units are expected to differ to some degree in spectral response.
