f_CO.NCA.NCL R Documentation

OC Curve for AOQL and LPTD relation

Description

Given an AOQL, LPTD, sample size and acceptance number, return the plot the OC curve and producer and consumer risk.The calculation uses the binomial trials. Applies for attribute plans.

Usage

f_CO.NCA.NCL(NCA,NCL,n,c)

Arguments

NCA

Specific AOQL value
NCL

Specific LPTD value
n

sample size
c

acceptance number

Details

Functionn stops if any value is missing

Value

NCA

Specific AOQL value
NCL

Specific LPTD value
n

sample size
c

acceptance number
beta

consumer risk
alpha

producer risk

Author(s)

Erick Marroquin

References

Examples
