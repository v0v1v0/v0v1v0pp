pnw_palettes R Documentation

Complete list of palettes

Description

Use names(pnw_palettes) to view list of palette names. Currently: Starfish, Shuksan, Bay, Winter, Lake, Sunset, Shuksan2, Cascades, Sailboat, Moth, Spring, Mushroom, Sunset2, and Anemone.

Usage

pnw_palettes

Format

An object of class list of length 14.
