unload_dataset R Documentation

unload_dataset

Description

Unload the dataset from a model

Usage

unload_dataset(model)

Arguments

model

(Model) Pharmpy model

Value

(Model) Pharmpy model with dataset removed

Examples

## Not run: 
model <- load_example_model("pheno")
model <- unload_dataset(model)
model$dataset is NULL

## End(Not run)
