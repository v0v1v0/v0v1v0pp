cloudwatchlogs_delete_delivery R Documentation

Deletes s delivery

Description

Deletes s delivery. A delivery is a connection between a logical delivery source and a logical delivery destination. Deleting a delivery only deletes the connection between the delivery source and delivery destination. It does not delete the delivery destination or the delivery source.

See https://www.paws-r-sdk.com/docs/cloudwatchlogs_delete_delivery/ for full documentation.

Usage

cloudwatchlogs_delete_delivery(id)

Arguments

id

[required] The unique ID of the delivery to delete. You can find the ID of a delivery with the describe_deliveries operation.
