SNPs R Documentation

A data frame of 100 artificial SNPs with their chromosomes and positions on the genome

Description

A data frame with 100 rows and 3 columns.

Usage

data(SNPs)

Format

Column names:

Name

SNP name

Position

Position of SNP on the genome

Chr

Chromosome of the SNP

See Also

genes, genotypes

Examples

data(SNPs)
print(SNPs[1:5,])
