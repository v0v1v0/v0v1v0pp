pollimetry_dataset R Documentation

Body size and intertegular distance measurements of ~4500 pollinating insect specimens.

Description

A dataset containing the body size in dry weight (mg), intertegular distance (mm), body length (mg) and metadata for 4438 measured bee and hoverfly specimens.

Usage

pollimetry_dataset

Format

An object of class data.frame with 4434 rows and 25 columns.

Details

@references Kendall et al. (2018) Pollinator size and its consequences: Predictive allometry for pollinating insects. <doi:10.1101/397604>
