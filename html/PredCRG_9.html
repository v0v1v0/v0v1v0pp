test R Documentation

Test dataset.

Description

A test dataset containing 54 circadian protein sequences collected from literature. This dataset has been used as an independent test dataset for assessing the predition accuracy of PredCRG model.

Usage

data("test")

See Also

PredCRG, PredCRG_Enc, PredCRG_data

Examples

data(test)
PredCRG(test[1:10])
