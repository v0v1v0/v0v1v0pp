Mnorm R Documentation

Matrix Norm

Description

Matrix Norm

Usage

Mnorm(X, k = 2)

Arguments

X

matrix

k

norm number

Details

returns the largest singular value of the matrix or vector

Value

Scalar Norm

Note

if k=1, absolute value; k=2 2-norm (rms); k>2, largest singular value.

Author(s)

Jonathan M. Lees<jonathan.lees@unc.edu>

Examples

x = runif(10)

Mnorm(x, k = 2)
