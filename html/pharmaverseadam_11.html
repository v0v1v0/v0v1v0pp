adlbhy R Documentation

Analysis of Lab Hy's Law

Description

A template of the ADLBHY dataset

Usage

adlbhy

Format

A data frame with 14 columns:

STUDYID

Study Identifier

USUBJID

Unique Subject Identifier

TRT01A

Actual Treatment for Period 01

PARAMCD

Parameter Code

LBSEQ

Sequence Number

ADT

Analysis Date

AVISIT

Analysis Visit

ADY

Analysis Relative Day

AVAL

Analysis Value

ANRHI

Analysis Normal Range Upper Limit

CRIT1

Analysis Criterion 1

CRIT1FL

Criterion 1 Evaluation Result Flag

AVALC

Analysis Value (C)

PARAM

Parameter

Source

Generated from admiral package (template ad_adlbhy.R).

References

None

Examples

data("adlbhy")
