get_wave R Documentation

Retrieve panel_data metadata

Description

get_id(), get_wave(), and get_periods() are extractor functions that can be used to retrieve the names of the id and wave variables or time periods of a panel_data frame.

Usage

get_wave(data)

get_id(data)

get_periods(data)

Arguments

data

A panel_data frame

Value

A panel_data frame

Examples


data("WageData")
wages <- panel_data(WageData, id = id, wave = t)
get_wave(wages)
get_id(wages)
get_periods(wages)
