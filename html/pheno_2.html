date2jul1 R Documentation

Converts string date to Julian date

Description

Converts a string date "DD.MM.YYYY" into a Julian day of year (DOY).

Usage

date2jul1(d)

Arguments

d

Date as charater string 'DD.MM.YYYY'.

Value

doy

Day of year as integer.
year

Year as integer.

Author(s)

Joerg Schaber

Examples

	date2jul1('31.05.1970')
