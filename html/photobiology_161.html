checkMspctVersion R Documentation

Check that the "mspct.version" attribute is set

Description

Function to check the "mspct.version" attribute of an existing generic_spct object.

Usage

checkMspctVersion(x)

Arguments

x

a generic_mspct object

Value

numeric value

Note

if x is not a generic_mspct object, NA is returned, and if it the attribute is missing, zero is returned with a warning.
