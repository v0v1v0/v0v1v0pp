ppgmmga-class R Documentation

Class 'ppgmmga'

Description

An S3 class object for ppgmmga algorithm

Objects from the class

Object can be created by calls to the ppgmmga function.

Values

data

The input data matrix.

d

The dimension of the projection subspace.

approx

The type of approximation used for computing negentropy.

GMM

An object of class 'densityMclust' containing the Gaussian mixture density estimation. See densityMclust for details.

GA

An object of class 'ga' containing the Genetic Algorithm search. See ga for details.

Negentropy

The value of maximised negentropy.

basis

The matrix basis of the projection subspace.

Z

The matrix of projected data.
