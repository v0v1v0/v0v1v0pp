Description

Usage

opsworks_update_rds_db_instance(
  RdsDbInstanceArn,
  DbUser = NULL,
  DbPassword = NULL
)

Arguments

RdsDbInstanceArn

[required] The Amazon RDS instance's ARN.
DbUser

DbPassword
