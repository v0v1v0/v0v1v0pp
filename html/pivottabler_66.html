typeSafeUnion R Documentation

Union two vectors without changing their data types.

Description

typeSafeUnion preserves data types in a way that the base::union function does not, e.g. for Date values.

Usage

typeSafeUnion(x, y, dedupe = FALSE)

Arguments

x

First set of values.
y

Second set of values.
dedupe

Specify TRUE to remove duplicate values.

Value

A vector containing the union of x and y
