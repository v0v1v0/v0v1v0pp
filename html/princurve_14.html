start_circle R Documentation

Generate circle as initial curve

Description

The starting circle is defined in the first two dimensions, and has zero values in all other dimensions.

Usage

start_circle(x)

Arguments

x

The data for which to generate the initial circle

Examples

## Not run: 
x <- cbind(
  rnorm(100, 1, .2),
  rnorm(100, -5, .2),
  runif(100, 1.9, 2.1),
  runif(100, 2.9, 3.1)
)
circ <- start_circle(x)
plot(x)
lines(circ)

## End(Not run)
