fmeta_matches R Documentation

Check that the f_meta file contains a column aligned to each omicsData objects

Description

Check that the f_meta file contains a column aligned to each omicsData objects

Usage

fmeta_matches(omicsData_objects, f_meta)

Arguments

omicsData_objects

A list of omicsdata objects containing sample information matching that in f_meta
f_meta

passed from as.multiData

Value

A list, each element of which contains a character vector of column name matches in f_meta for each omicsData object.
