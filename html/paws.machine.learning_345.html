lexmodelbuildingservice_get_builtin_intents R Documentation

Gets a list of built-in intents that meet the specified criteria

Description

Gets a list of built-in intents that meet the specified criteria.

See https://www.paws-r-sdk.com/docs/lexmodelbuildingservice_get_builtin_intents/ for full documentation.

Usage

lexmodelbuildingservice_get_builtin_intents(
  locale = NULL,
  signatureContains = NULL,
  nextToken = NULL,
  maxResults = NULL
)

Arguments

locale

A list of locales that the intent supports.
signatureContains

Substring to match in built-in intent signatures. An intent will be returned if any part of its signature matches the substring. For example, "xyz" matches both "xyzabc" and "abcxyz." To find the signature for an intent, see Standard Built-in Intents in the Alexa Skills Kit.
nextToken

A pagination token that fetches the next page of intents. If this API call is truncated, Amazon Lex returns a pagination token in the response. To fetch the next page of intents, use the pagination token in the next request.
maxResults

The maximum number of intents to return in the response. The default is 10.
