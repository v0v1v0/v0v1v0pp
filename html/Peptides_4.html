aaList R Documentation

Return a vector with the 20 standard amino acids in upper case

Description

This function returns a vector with the 20 standard amino acids in upper case.

Usage

aaList()

Value

A character vector with the 20 standard amino acids in upper case.

Author(s)

Richel Bilderbeek <richel@richelbilderbeek.nl>

References
