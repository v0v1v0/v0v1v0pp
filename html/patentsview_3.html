get_endpoints R Documentation

Get endpoints

Description

This function reminds the user what the 7 possible PatentsView API endpoints are.

Usage

get_endpoints()

Value

A character vector with the names of the 7 endpoints. Those endpoints are:

  • assignees

  • cpc_subsections

  • inventors

  • locations

  • nber_subcategories

  • patents

  • uspc_mainclasses
