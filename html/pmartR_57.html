get_fdata_cname R Documentation

Fetch the f_data column name

Description

This function returns the name of the column in f_data that contains the names of the samples.

Usage

get_fdata_cname(omicsObject)

Arguments

omicsObject

an object of the class 'pepData', 'proData', 'metabData', 'lipidData', 'nmrData', 'statRes', or 'trelliData', usually created by as.pepData, as.proData, as.metabData, as.lipidData, as.nmrData, imd_anova, or as.trelliData respectively.

Value

a character string describing f_data cname
