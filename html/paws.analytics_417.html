firehose_put_record R Documentation

Writes a single data record into an Amazon Firehose delivery stream

Description

Writes a single data record into an Amazon Firehose delivery stream. To write multiple data records into a delivery stream, use put_record_batch. Applications using these operations are referred to as producers.

See https://www.paws-r-sdk.com/docs/firehose_put_record/ for full documentation.

Usage

firehose_put_record(DeliveryStreamName, Record)

Arguments

DeliveryStreamName

[required] The name of the delivery stream.
Record

[required] The record.
