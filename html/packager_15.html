get_check_status R Documentation

Retrieve Check Status From a Log File

Description

Searches a typical R CMD check like ‘xxx.Rcheck/00check.log’ for Errors, Warnings and Notes.

Usage

get_check_status(path)

Arguments

path

Path to the log file.

Value

A list :

status

A list of

notes

The number of NOTEs

warnings

The number of WARNINGs

error

The number of ERRORs

log

A list of

notes

The log entries for NOTEs

warnings

The log entries for WARNINGs

error

The log entries for ERRORs
