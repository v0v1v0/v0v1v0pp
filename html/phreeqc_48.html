phrGetDumpStringsOn R Documentation

Retrieve the current value of the dump strings switch.

Description

Retrieve the current value of the dump strings switch.

Usage

phrGetDumpStringsOn()

Value

TRUE if output as defined by the DUMP data block are currently being stored as a character vector.
