rs_onco_irecist R Documentation

Disease Response (iRECIST) for Oncology

Description

A SDTM RS dataset using iRECIST. The dataset contains just a few patients. It is intended for vignettes and examples of ADaM dataset creation.

Usage

rs_onco_irecist

Format

An object of class tbl_df (inherits from tbl, data.frame) with 376 rows and 19 columns.
