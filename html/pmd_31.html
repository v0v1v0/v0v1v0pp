spmeinvivo R Documentation

A peaks list dataset containing 9 samples from 3 fish with triplicates samples for each fish from LC-MS.

Description

A peaks list dataset containing 9 samples from 3 fish with triplicates samples for each fish from LC-MS.

Usage

data(spmeinvivo)

Format

A list with 4 variables from 1459 LC-MS peaks:

mz

mass to charge ratios

rt

retention time

data

intensity matrix

group

group information
